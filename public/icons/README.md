# PWA Icons

This directory contains the app icons required for the Progressive Web App.

## Required Icons

You need to create the following icon files:

1. **icon-192x192.png** - 192x192 pixels
   - Used for Android home screen and app launcher
   - Should be a square icon with your app logo

2. **icon-512x512.png** - 512x512 pixels
   - Used for Android splash screen and high-resolution displays
   - Should be a square icon with your app logo

3. **apple-touch-icon.png** - 180x180 pixels
   - Used for iOS home screen icon
   - Should be a square icon with your app logo
   - iOS will automatically add rounded corners and effects

## Icon Design Guidelines

- Use a simple, recognizable logo or icon
- Ensure the icon is clear and visible at small sizes
- Use high contrast colors for better visibility
- Avoid text in icons (it becomes unreadable at small sizes)
- Test icons on both light and dark backgrounds
- Icons should be square (iOS will add rounded corners automatically)

## Creating Icons

You can create these icons using:
- Design tools like Figma, Adobe Illustrator, or Canva
- Online icon generators
- Image editing software (Photoshop, GIMP, etc.)
